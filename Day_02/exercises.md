## Day 02
1. What's VirtualBox, and why use it for learning?
    - It it a type 2 hypervisor that runs on top of the host system and helps create guest VM's
2. Step-by-step: How do you create a VM?
    - Open Vbox, select ISO, select configurations for Hardware, user settings, Install
3. Why snapshots in virtualization?
    - They are used to create a phase or as a checkpoint that can be used to bring OS to that state.
4. Basic network check in a VM?
    - Can be done using ping google.com and if still not working check the NAT settings
5. Quick boot overview?
    - BIOS -> GRUB -> Kernel -> Services -> Login
